# Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Migrations

- Make migration: `npm run typeorm:generate-migration migrations/<name_of_migration>`
- Run migrations: `npm run typeorm:run-migrations`
